# Celebrate Festival - Frontend

Frontend part of the Celebrate Festival application built with React 19, TypeScript, Vite, and Tailwind CSS.

## Features

- Modern React 19 with TypeScript
- Tailwind CSS 4 for styling
- React Router DOM for navigation
- Beautiful animations and transitions
- Responsive design for all devices
- Integration with backend API

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- React 19
- TypeScript
- Vite 7
- Tailwind CSS 4
- React Router DOM 7
- Axios
- lunar-javascript
